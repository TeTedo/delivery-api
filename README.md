# API URI
| 기능 |  Method | URI              |
|------|---------|------------------|
| 음식 전체 조회 | GET | /foods           |
| 음식 단일 조회 | GET | /foods/{food-id} |
| 음식 등록 | POST | /foods           |
| 음식 전체 수정 | PUT | /foods           |
| 음식 부분 수정 | PATCH | /foods           |
| 음식 삭제 | DELETE | /foods/{food-id} |
